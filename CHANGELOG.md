# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.3.0] - 2026-07-19

### Added

- Drag-and-drop reordering of array elements.

## [1.2.0] - 2026-07-16

### Added

- File explorer tree view for browsing notes.

### Fixed

- Bottom row actions overflowing on narrow screens.
- Tree disclosure (expand/collapse) visibility.

## [1.1.0] - 2026-07-16

### Removed

- Application history feature.
- The concept of trash.
